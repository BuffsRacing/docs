# Hardware

## ESP32

### ESP32-CAM

RTSP / Save to Disk

#### RTSP Streamer

Starts mDNS responder and can be accessed via `ralphiecam-00.local`. Every camera that is added, automatically increments that number.

#### Save to Disk

Saves video to SD-Card at full HD resolution
